Welcome To QTV Guitar Shop

Sau khoảng thời gian gián đoạn kinh doanh vì đại dịch Covid 19, chúng tôi rất vui mừng thông báo rằng Showroom tại Hồ Gươm plaza, Trần Phú, Hà Đông sẽ mở cửa trở lại bắt đầu từ thứ hai ngày 04/05/2020. Các bạn có thể trực tiếp ghé thăm và trải nghiệm các loại nhạc cụ tại Showroom.

Tuy dịch bệnh đã được kiểm soát nhưng vẫn còn tiềm ẩn rất nhiều nguy cơ lây nhiễm trong cộng đồng, nên đừng quên đeo khẩu trang khi ra đường và rửa tay thường xuyên các bạn nhé.

About This Website

This website is built as part of my Front-end coding bootcamp at TechMaster ( Ha Noi, Vietnam ), not for any commercial purposes. Content in this website ( photos, videos, articles, ...) is collected on the internet and doesn't belong to me, I have only created UI/UX design and coding part. You can find UI/UX design at : https://www.figma.com/file/YhjxIz2P6KEcUaTrtCkQuu/QTV-Guitar-Shop-Project?node-id=145%3A593 ( need to create a Figma account if you don't have one )

In this project, some pages are written by using pure Javascript and all other pages are written by using Jquery. I have done that just for practicing both Javascript and Jquery( a very popular library ). For self-training and challenging myself I haven't used any CSS or Javascript framework and library ( except Jquery ).

Special thanks to Ms. Pham Thi Yen, Mr. Le Chi Linh and Mr. Luu The Vinh for providing guidance and feedback to help me complete this project.

If you have any questions, feel free to contact me by email: quangtuanvu1991@gmail.com. Thanks.
